OPTIONS
=======

**--help, -h**
--------------

Displays short usage help.

**--version, -v**
-----------------

Displays version information.

**--vhost** \<host-name\>
-------------------------

Sets virtual host name.  This option is mandatory.

**--ip** [=0.0.0.0]
-------------------

IP address to bind to.  The default is to bind to all interfaces.

**--repo** [=.]
---------------

Path to repository.

**--port** [=8000]
------------------

Post to listen on.
